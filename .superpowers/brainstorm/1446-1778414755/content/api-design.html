API 接口设计

RESTful，全部走 Next.js API Routes

🔥 趋势相关
GET /api/trends
查询: ?category=&sort=heat&page=
返回: 分页话题列表

GET /api/trends/:id
返回: 话题详情 + 热度曲线 + AI预判
🎨 生成相关
POST /api/generate
Body: {topicId, modelId, type, prompt, params}
返回: {taskId, status:"queued"}

GET /api/generate/:taskId
返回: {status, progress, result?}
status: queued→processing→done|failed
📦 内容库
GET /api/library
查询: ?type=&status=&page=

GET /api/library/:id
返回: 内容详情 + 发布建议

DELETE /api/library/:id
👤 用户 & 积分
GET /api/user/me
返回: 用户信息 + 积分余额

GET /api/user/transactions
积分流水记录

POST /api/user/recharge
创建充值订单 → 返回支付链接
🤖 模型信息
GET /api/models
返回: 可用模型列表 + 类型 + 消耗积分
🔐 认证 (NextAuth)
POST /api/auth/signup
POST /api/auth/signin
GET /api/auth/session
邮箱 + 微信扫码 两种登录方式

统一响应格式

// 成功
{ "code": 0, "data": { ... }, "message": "ok" }

// 失败
{ "code": 40001, "data": null, "message": "积分不足" }

// 分页
{ "code": 0, "data": { "list": [...], "total": 100, "page": 1, "pageSize": 20 } }

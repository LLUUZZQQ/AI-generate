技术选型

每个层面的工具选择及理由

🖥️ 前端
Next.js 15 — React全栈框架
Tailwind CSS — 原子化样式
shadcn/ui — 高质量组件库
Zustand — 轻量状态管理
React Query — 数据请求+缓存
⚙️ 后端
Next.js API Routes — 前后端一体
Prisma ORM — 类型安全数据库操作
NextAuth.js — 认证（邮箱+微信+手机）
Zod — 输入校验
🗄️ 数据层
PostgreSQL — 主数据库
Redis — 任务队列 + 缓存
BullMQ — Node.js 任务队列
OSS/S3 — 图片视频存储
🤖 AI + 外部服务
OpenAI SDK — DALL-E / Sora
Replicate — Stable Diffusion
国产 API — 通义万相/文心一格/即梦
Puppeteer — 抖音趋势抓取
微信/支付宝 — 支付

核心选型理由

Next.js 单体 你一个人开发，前后端分离没必要。Next.js 一套代码搞定页面+API+认证
Prisma 类型安全的 ORM，自动生成 TypeScript 类型，改 schema 改代码都有智能提示
BullMQ + Redis Node 生态最成熟的任务队列。趋势抓取和 AI 生成都是分钟级任务，必须异步
shadcn/ui 不是 npm 包，代码直接拷进项目。你完全控制源码，后续定制不打架
模型层抽象 所有 AI 模型通过统一接口调用：(model, prompt, params) → result。新增一个模型只需加配置，不改业务代码

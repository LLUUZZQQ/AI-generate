页面结构 & 路由设计

8个核心页面，覆盖完整用户旅程

🏠 / — 首页/落地页
产品介绍、核心功能展示、定价方案
未登录用户 → 引导注册
已登录 → 跳转趋势页
🔥 /trends — 趋势发现
实时热门话题瀑布流
按热度/分类/时间筛选
每个卡片显示：话题名、热度值、趋势箭头↑↓、热点预判标签
📊 /trends/[id] — 话题详情
热度变化曲线图
AI 趋势预判分析
关联内容展示
一键「以此话题生成」按钮
🎨 /generate — 内容生成
选择话题 → 选择模型 → 写提示词
支持图片/视频两种模式切换
显示本次消耗积分
可调整参数：分辨率、风格、时长
📦 /library — 我的内容库
所有生成历史，按时间/话题排列
每条显示：缩略图、状态、话题、模型、时间
操作：下载、查看发布建议、删除
📝 /library/[id] — 内容详情+发布建议
大图/视频预览 + 下载
AI 发布建议面板：
最佳时段、推荐配文、热门标签、BGM推荐
👤 /dashboard — 个人中心
积分余额 + 消费统计
生成数量统计
快速入口：趋势页 / 生成页
⚙️ /settings — 设置 + 充值
账户信息管理
积分充值（微信/支付宝）
API Key 管理（高级用户）
消费记录明细

用户核心旅程

  进入首页 → 注册/登录(送20积分) → 浏览趋势页🔥 → 看到感兴趣话题 →
  查看话题详情📊 → 点击「以此话题生成」→ 选择模型+写提示词 → 提交生成 →
  等待完成(进度条) → 查看结果+发布建议📝 → 重复以上 → 积分不足 → 充值 → 继续
